Jawaban:

Ada di penjelasan

Penjelasan dengan langkah-langkah:

bagian II

1. 3,45 = 345/100 = 69/20

dijadikan pecahan biasa kemudian pembilang Dan penyebut sama-sama dibagi 5.

2. 3,8 + 3 6/15 = 38/10 + 3 2/5 = 19/5 + 17/5 = 36/5

3.8 diubah menjadi pecahan biasa menjadi 38/10, kemudian disederhanakan dg Cara pembilang Dan penyebutnya sama-sama dibagi 2

bagian 6/15 dari 3 6/15 ini disederhanakan dg Cara penyebut Dan pembilangnya sama-sama dibagi 2, sehingga jadinya 3 2/5.

3. 6/15 = 2/5

2 dibagi 5, maka hasilnya 0.4

4. 36 % + 4/5 x 100 % = 36 % + 80 % = 116 %

5. 64 % - 0.57 = 64 % - 57/100 x 100 % = 64 % - 57 % = 7 %

6. 6/8 = 3/4 (pembilang Dan penyebut sama2 dibagi 2), maka

3 dibagi 4 = 0,75

7. 3 2/5 - 1 2/3 = 17/5 - 5/3 = 17x3/5x3 - 2x5/3x5 = 51/15 - 10/15 = 41/15 = 2 11/15

8. 14,06 - 5,68 = 8,36

9. 335 % + 175 % = 510 % = 510/100 = 5,1

10. 2 2/5 - 1,5 = 12/5 - 15/10 = 12x2/5x2 - 15/10 = 24/10 - 15/10 = 9/10 = 0,9

bagian III

1. air yg tumpah = 6,5 - 2 1/5 = 6,5 - 2,2 = 4,3 liter

2. 2 1/2 - 1,25 = 2.5 - 1,25 = 1,25 kg

3. minyak goreng yg harus ditambah = 1 - 1/5 = 1 - 0,2 = 0,8 bagian

25. kunci belajar question tag yang gampang


Kunci belajar Question Tag :
~ Memahami berbagai pengecualian di dalam Question Tag seperti : 
a) Untuk I am... question tagnya aren't I?
b) Untuk kalimat perintah/larangan question tag nya "will you"
c) Untuk "Let's....." question tagnya shall we ?
~ Ingat kunci penting question tag yakni apabila kalimat positif maka question tagnya negatif dan sebaliknya
~ Ingat tentang macam personal pronoun

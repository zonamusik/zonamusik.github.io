5. Question tag .........? Lengkapilah dengan question tag


Answers

1. Your name is Salma, isn't it?

2. Those men spoke English very well, didn't they?

3. You and I are friends, aren't we?

4. That car moves very fast, doesn't it?

5. Rini's boy friends come here, don't they?

6. My parents won't go to Bali, will they?

7. Marisa has come here on time, hasn't she?

8. They should study hard now, shouldn't they?

9. Father reads newspaper every morning, doesn't he?

10. We don't always go to the swimming pool every Sunday, do we?

---------

Explanation

Question tag adalah sebuah klausa interogatif singkat yang diletakkan di ujung sebuah pernyataan/perintah. Question tag mengubah sebuah pernyataan menjadi pertanyaan dan sebuah kalimat perintah menjadi permintaan.

Question tag biasanya dipakai untuk meminta penegasan atas suatu pernyataan.

Question tag selalu berlawanan dengan kalimat utamanya.

Jika kalimat utama positif, maka tag negatif, dan sebaliknya.

Konstruksi Question Tag adalah : kata kerja bantu/modal + subject pronoun.

Jika tagnya negatif, kata kerja bantu+not selalu dikontraksi (disingkat).

14. Dengan menggunakan tag ini kita juga bisa membuat list tanpa simbol atau angka didepan keterangan list adalah a. Tag UL b. Tag OL c. Tag DL d.Tag Do​


Cara 1:
Dengan menggunakan tag <DL>, kita bisa membuat list tanpa simbol atau angka didepan keterangan list. Namun perlu diperhatikan bahwa tag <DL> digunakan untuk membuat description list.Cara 2:
Dengan menggunakan tag <UL> atau tag <OL> yang diberi styling “list-style: none;”, kita juga bisa membuat list tanpa simbol atau angka didepan keterangan list.

 

Pembahasan

Untuk membuat list pada dokumen HTML, kita dapat menggunakan tag <UL>, <OL>, atau <DL>.

UL adalah singkatan dari unordered list, artinya daftar yang tidak diurutkan berdasarkan angka. OL memiliki kepanjangan ordered list, artinya daftar yang terurut berdasarkan angka.DL adalah singkatan dari definition/description list, artinya daftar yang berisi definisi/deskripsi.

Berbeda dengan tag <UL> dan <OL> yang membutuhkan tag <LI> untuk setiap item dalam list, tag <DL> membutuhkan tag <DT> (term) dan tag <DD> (description).

Jadi, cara pertama untuk membuat list tanpa simbol atau angka didepan keterangan list adalah dengan tag <DL>.

Kita juga bisa membuat list tanpa simbol atau angka dengan tag <UL> dan <OL>, dengan menambahkan kode style “list-style: none;” pada kedua tag tersebut.

Contoh perbedaan dan tampilan kode-kode tag tersebut dapat dilihat pada gambar.

20. There is no price tag the underlined words has a synonym meaning with....


Dalam bahasa Inggris dikenal juga synonym atau yang dalam bahasa Indonesia disebut sinonim. Underline word dari soal di atas adalah kata price tag yang memiliki sinonim value.

Pembahasan:

Makna leksikal merupakan kata yang bermakna asli, atau makna sesungguhnya dari suatu bentuk kebahasaan. Karena makna dari suatu kalimat tidaklah tunggal, maka makna leksikal dapat dibedakan menjadi beberapa jenis, antara lain:

• Sinonim

Sinonim disebut juga dengan persamaan kata atau kata yang memiliki makna sama.

• Antonim

Antonim merupakan lawan kata atau kata yang memiliki makna berkebalikan.

• Homonim

Homonim disebut juga dengan persamaan bunyi.

• Hiponim

Hiponim merupakan suatu kata yang mewakili kelompok kata tertentu.

• Meronim

Meronim merupakan bagian yang mewakili sesuatu secara menyeluruh.

Makna gramatikal merupakan makna suatu kata yang dapat muncul setelah diubah ke dalam kalimat. Makna gramatikal dapat disebabkan oleh:

• Imbuhan (afikasi)

• Reduplikasi (pengulangan kata)

• Pembentukan kata

• Perubahan komposisi kata

• Klausa

• Pembentukan kalimat

29. jelaskan perbedaan antara tag di HTML dan tag di CSS!


Jawaban:

HTML adalah singkatan dari HyperText Markup Language , yaitu markup language yang digunakan untuk membuat suatu halaman web, sehingga HTML dapat menampilkan berbagai informasi baik tulisan maupun gambar pada sebuah web browser.

CSS adalah singkatan dari Cascading Style Sheet, yaitu suatu aturan yang digunakan untuk mengatur tampilan maupun format dari suatu halaman web, sehingga tampilan web akan menjadi lebih teratur.

--semoga membantu--

Jawaban:

HTML biasanya menggunakan tag-tag tertentu yang berada dalam tanda kurung <>. Sebagian besar tag pada HTML adalah tag yang berpasangan, contoh : jika kita menggunakan <head> harus ditutup dengan </head>, atau jika kita menggunakan <p> harus ditutup dengan </p>. Tapi ada juga beberapa tag yang tidak berpasangan, contohnya adalah <img>.

Sedangkan untuk CSS ada 3 cara penulisan, yakni internal, inline dan eksternal.

Contoh penulisan inline :

<h1 style="color:red">Judul</h1>

Internal CSS diletakkan di dalam tag <style>, dan biasanya tag <style> ditempatkan pada bagian tag <head>  

Contoh penulisan internal :

<style type="text/css">

.box{

 background: blue;

 padding: 10px;

}

</style>

Sementara untuk eksternal, antara file html dan css dipisah,berikut cara menghubungkan file css dengan html :  

<link rel="stylesheet" type="text/css" href="file.css">

37. pengertian tag pembuka, tag penutup dan tag berdiri sendiri di dalam html????


Mata pelajaran : TI

Tema : HTML

Kode soal : 11

Pengertian dari tag pembuka, tag penutup dan tag berdiri sendiri

Tag pembuka = tag yang digunakan untuk membuka deklarasi dari suatu hal yang akan dibuat.

Contoh : <html>, <head>, <body>

Tag penutup = tag yang digunakan sebagai penutup dari deklarasi terhadap suatu hal yang dibuat.

Contoh : </html>, </head>, </body>

Tag berdiri sendiri = tag yang berdiri sendiri, biasanya tidak memiliki pasangan dalam mendeklarasikannya.

Contoh : <br/>

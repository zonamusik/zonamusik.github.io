Revolusi bumi adalah peredaran bumi mengelilingi matahari.

Akibat dari revolusi bumi adalah:

Perbedaan Lama Waktu Siang dan Malam.Perubahan Rasi Bintang.Gerak Semu Tahunan Matahari.Adanya Perubahan Musim.Ditetapkannya Kalender Masehi.

Rotasi bumi adalah perputaran bumi pada sumbunya.

Akibat dari rotasi bumi adalah:

Terjadinya Siang dan Malam. Gerak Semu Harian Matahari.Perbedaan Waktu.Perbedaan Percepatan Gravitasi Bumi.Pembelokan Arah Arus Laut.Jadi bisa disimpulkan bahwa nomor 2 dan nomor 5 adalah dampak dari rotasi bumi dan bukan revolusi bumi.

Jadi jawaban yang benar adalah nomor 1, 3, dan 4.

[tex]\bf \purple{ {a}^{n} = a \times a \times a... \times a}[/tex]

Penjelasan:

Dua pertanyaan tersebut menanyakan tentang mengubah kalimat aktif menjadi kalimat pasif.

Cara mengubahnya adalah pertama, geser objek tindakan ke awal kalimat (posisi subjek).  Lalu, bah kata kerja-nya. Jika kamu ingin mempertahankan aktor, subjek asli, geser ke akhir kalimat dan masukkan kata di depannya.

39. untuk penggunaan apa kode Goo jelaskan​


Penjelasan:

G00 : Gerakan bebas tanpa pemakanan atau pergerakan cepat

G01 : Gerakan pemakanan lurus yang di sertai dengan kecepatan pemakanan ( feeding )

G02 : Gerakan melengkung atau pembuatan radius ( R ) yang searah jarum jam (CW).

G03 : Gerakan melengkung atau pembuatan radius ( R ) yang berlawanan jarum jam (CCW).

G04 : Gerakan berhenti sesaat ( dwell ).

G25 : Memanggil sub program

G41 : untuk pembubutan diameter dalam

G42 : untuk pembubutan diameter luar

G64 : Mematikan arus step motor

G65 : Operasi disket

G70 : perintah finishing

G71 : perintah pemakanan berulang secara otomatis

G72 : perintah pemakanan berulang secara otomatis

G74 : perintah pengeboran dengan pemutusan tatal

G78 : Siklus pembuatan ulir ( thread )

G81 : Siklus pengeboran langsung

G82 : Siklus pengeboran dengan berhenti sesaat

G83 : Siklus pengeboran dengan penarikan tatal

G84 : Siklus pembubutan memanjang (Z)

G85 : Siklus perimeran

G86 : Siklus pembuatan alur

G88 : Siklus pembubutan melintang (X)

G89 : Siklus perimeran dengan waktu berhenti sesaat

G90 : Program Absolute

G91 : Program Inkrimental

G92 : pembuatan ulir ( thread )

G97 : Perputaran rpm stabil

G00 = gerakan cepat tanpa penyayatan

MIT App Inventor Cara membuat Aplikasi Android di MIT App Inventor cocok untuk anda yang mengerti koding namun belum ingin merogoh kocek. MIT App Inventor adalah aplikasi dengan basis open sources (gratis) yang dikelola oleh MIT, sebelumnya App Inventor dikelola oleh google. Menurut beberapa review, App Inventor ini sangat cocok untuk membuat animasi dan games android. di sini Aplikasi dibuat dengan "komponen" dan "blok" yang dipadukan dengan tampilan desain. Nah daripada penasaran, silahkan langsung coba ke webnya aja. Gratis kok.
Kelebihan MIT adalah Gratis, sederhana, mudah digunakan, dan cocok untuk kita yang bukan lulusan ilmu komputer (yang tidak mengerti bahasa pemrograman). Bahkan, menurut instanshift.com, MIT App Inventor telah menjadi tempat pembuat beberapa aplikasi sukses. jadi jangan ragu untuk belajar cara membuat aplikasi android gratis di sini. 
Buat sobat yang ingin mendalami MIT App Inventor, silahkan membaca tutorial berikut Cara Membuat Aplikasi Android dengan App Inventor [Ebook Tutorial]

25. Bagaimana cara membuat aplikasi pada android? ​


Pake Android Studio

Caranya;

1. Create new project

- Buka android studio dan pilih start a new android studio project

- Isikan nama aplikasinya dan pilih lokasi penyimpanan project android studionya.

- Pilih minimum OS Android yang dibutuhkan untuk menjalankan aplikasimu

- Pilih empty activity

- Beri nama activity yang akan dibuat lalu klik Finish

2. Tampilan Environment Android Studio

- Kamu cukup focus pada main activity dan activity_main.xml

3. Melihat Tampilan Android

- Buka file activity_main.xml

4. Jalankan aplikasi android

- Klik tombol play berwarna hijau

- Pilih dimana kamu akan menjalankan aplikasi androidnya

- Untuk yang punya smartphone android, pergi ke developer options di menu settings android-nya → Lalu aktifkan opsi USB Debugging → Tancapkan smartphone ke komputermu via kabel USB → Jika muncul peringatan di android mengenai perizinan USB Debugging, klik saja ALLOW → Install driver smartphone android di komputermu → (Kembali ke Android Studio) → Maka device android-mu akan muncul di dialog tadi di bagian Connected Devices. Klik device android-mu lalu klik OK, tunggu hingga proses compiling selesai dan aplikasi android-mu akan berjalan di smartphone-mu.

5. Selesai

Android Studio sangat user-friendly. Dengan kesungguhan dan mau belajar kamu pasti bisa menguasainya dengan baik meskipun tidak mahir coding.

Semoga membantu

32. bagaimana caranya menyembunyikan aplikasi tanpa aplikasi di handphone android ?


Cara menyembunyikan aplikasi di Android bisa dilakukan dalam 10 detik.
1. Tinggal buka aplikasi atau menu pengaturan.
2. Setelah itu, tinggal pilih “Applications”.
3. Langkah selanjutnya, kamu hanya perlu pilih “Application Manager”.
4. Pilih tab “All” dan sentuh aplikasi apa aja yang ingin kamu sembunyikan.
5. Kemudian pilih “Disable”.

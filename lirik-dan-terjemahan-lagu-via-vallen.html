4. perbandingan uang via terhadap uang Vallen adalah 7 banding 4 jika selisih uang mereka Rp15.000 maka jumlah uang via dan uang Vallen adalah​


Jawab:

jumlah uang via dan uang vallen adalah Rp 55.000

Penjelasan dengan langkah-langkah:

selisih uang = 15.000

selisih perbandingan = 7 - 4 = 3

uang tiap perbandingan = 15.000 ÷ 3 = 5.000

uang via = 7 × 5.000 = 35.000

uang vallen = 4 × 5.000 = 20.000

uang total = 35.000 + 20.000 = 55.000

8. Perbandingan uang via terhadap uang vallen adalah 3 :4 jika selisih uang mereka 2. 500. 000,00, maka jumlah uang via dan uang vallen adalah.


Jumlah uang mereka

= (3+4)/(4-3) × 2.500.000

= 7/1 × 2.500.000

= Rp 17.500.000,00

23. Perbandingan uang via terhadap uang vallen adalah 3:4. Jika selisih uang mereka Rp 2. 500. 000,00 maka jumlah uang Via dan vallen adalah.


Jawaban singkat:

Rp 17.500.000,00

Penjelasan dg langkah²:

Uang Via = [tex]\frac{Nilai.Perbandingan.Via}{(Selisih.Perbandingan)}[/tex] x selisih uang mereka

Uang Via = [tex]\frac{3}{(4-3)}[/tex] x Rp 2.500.000,00

Uang Via = Rp 7.500.000,00

Uang Valen = [tex]\frac{Nilai.Perbandingan.Valen}{(Selisih.Perbandingan)}[/tex] x selisih uang mereka

Uang Valen = [tex]\frac{4}{(4-3)}[/tex] x Rp 2.500.000,00

Uang Via = Rp 10.000.000,00

Jumlah uang Via dan Valen = Rp 7.500.000,00 + Rp 10.000.000,00

Jumlah uang Via dan Valen = Rp 17.500.000,00

DIKETAHUI Perbandingan = Via : Vallen = 3 : 4, danSelisih = Rp 250.000,00.DITANYAJumlah uang keduanya = ...?CARATENTUKAN JUMLAH UANG VIA

[tex]Via = \frac{3}{4 - 3} \times 250.000 \\ Via = \frac{3}{1} \times 250.000 \\ Via = 750.000[/tex]

TENTUKAN JUMLAH UANG VALLEN

[tex]Vallen = Via + Selisih \\ Vallen = 750.000 + 250.000 \\ Vallen = 1.000.000[/tex]

TENTUKAN JUMLAH UANG KEDUANYA

[tex]= Via + Vallen \\ = 750.000 + 1.000.000 \\ = 1.750.000[/tex]

34. perbandingan uang via terhadap uang Vallen adalah 5:4 jika selisih uang mereka Rp 2.000.000.00 maka jumlah uang via dan uang Vallen adalah​


=================================

Diketahui :

Perbandingan uang via terhadap uang Vallen adalah 5:4

selisih uang mereka Rp 2.000.000

Ditanya :

jumlah uang via dan uang Vallen adalah

Jawaban :

Selisih perbandingan = 5 - 4 = 1

Uang Via = 5/1 × 2.000.000

Uang Via = 10.000.000

Uang Vallen = 4/1 × 2.000.000

Uang Vallen = 8.000.000

Jumlah uang via dan uang Vallen adalah

= 10.000.000 + 8.000.000

= 18.000.000

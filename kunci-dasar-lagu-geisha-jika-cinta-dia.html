28. Jika Umur Tono + Umur Cinta = 80, Umur Tono + Umur Geisha = 98, Umur Cinta + Umur Geisha = 94. Berapa Umur Tono, Cinta dan Geisha? 1. 28, 55, 19 2. 22, 58, 49 3. 42, 31, 63 4. 42, 38, 56 5. 38, 42, 56


Tono 42 tahun, Cinta 38 tahun, geisha 56 tahun
42+38= 80, 42+ 56= 98, 38+56=94
Jadi jawabannya adalah 4. 42, 38, 56 Umur Tono = T
Umur Cinta = C
Unur Geisha = G

Jika
T + C = 80 maka C = ( 80 - T )
T + G = 98 maka G = ( 98 - T )
dan
C + G = 94
( 80 - T ) + ( 98 - T ) = 94
80 + 98 - T - T = 94
178 - 2T = 94
- 2T = 94 - 178
- 2T = - 84
2T = 84
T = 84/2
T = 42 ( Jadi, umur Tono 42 tahun).

Umur Cinta
C = 80 - T
C = 80 - 42
C = 38 ( Umur Cinta 38 tahun)

Umur Geisha
G = 98 - T
G = 98 - 42
G = 56 (Umur Geisha 56 tahun)

Sehingga pilihan jawaban yang benar adalah
4. 42, 38, 56

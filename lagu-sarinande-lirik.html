14. apakah tema lagu sarinande? ​


Hal tersebut dalam lagu, diilustrasikan dengan seorang gadis meniup api di depan tungku sehingga mengepulkan asap.

Lagu Sarinande menceritakan tentang tradisi di Maluku, yaitu menguji gadis-gadis dalam keterampilan dalam bidang kewanitaan, sebelum sang gadis memasuki jenjang pernikahan.
Pada zaman dulu para gadis juga diuji keterampilan dalam hal memasak.
Dengan meniup api di depan tungku yang memerlukan kesabaran dan sikap rendah hati.
Sarinande merupakan seorang gadis yang beranjak dewasa. Ibu melihat Sarinande bengkak matanya, yang menandakan bahwa ia belum bisa dalam meniup api di tungku, "La asap api masuk dimata".
Dan esok harinya ia dapat mencobanya kembali.

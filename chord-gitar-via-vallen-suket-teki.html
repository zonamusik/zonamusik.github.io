Uang Via = [tex]\frac{Nilai.Perbandingan.Via}{(Selisih.Perbandingan)}[/tex] x selisih uang mereka

Uang Via = [tex]\frac{3}{(4-3)}[/tex] x Rp 2.500.000,00

Uang Via = Rp 7.500.000,00

Uang Valen = [tex]\frac{Nilai.Perbandingan.Valen}{(Selisih.Perbandingan)}[/tex] x selisih uang mereka

Uang Via = Rp 10.000.000,00

39. perbandingan uang vita dengan uang vallen adalah 3:4.jika selisih uang mereka rp 2.500.000,00,maka jumlah uang via dan vallen adalah


Pembahasan :

Perbandingan uang Vita dengan uang Vallen adalah 3 : 4. Jika selisih uang mereka Rp 2.500.000,00. Maka, jumlah uang vita dan vallen adalah ...

Penyelesaian :

Uang Vita :

[tex] = \frac{3}{4 - 3} \times 2.500.000\\ = \frac{7.500.000}{1} \\ = Rp \: 7.500.000,00[/tex]

Uang Vallen :

[tex] = \frac{4}{4 - 3} \times 2.500.000 \\ = \frac{10.000.000}{1} \\ = Rp \: 10.000.000,00[/tex]

Jadi, jumlah uang mereka (Vita dan Vallen) adalah 17.500.000,00 ✔

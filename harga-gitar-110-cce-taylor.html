Rp 9.600,00

Penjelasan dengan langkah-langkah:

Kenaikan = 20% × 8000

= 20/100 × 8000

= 160.000/100

= Rp 1.600,00

Harga setelah kenaikan = 8.000 + 1.600 = Rp 9.600,00

Jawaban:

Harga suatu barang, naik 20%. Jika harga sebelum kenaikan adalah Rp. 8.000,00, maka harga setelah kenaikan adalah Rp 9.600

Penjelasan dengan langkah-langkah:

Dimisalkan, harga setelah kenaikan → c

Maka:

c = 8.000 + (8.000 × 20%)

c = 8.000 + (8.000 × 20/100)

c = 8.000 + 1.600

c = Rp 9.600

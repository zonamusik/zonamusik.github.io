13/5 x = 520.000

x = 200.000

26. MOHON BANTUANYAHarga 3 HP merk A dan 2 HP merk B adalah Rp. 9.000.000,- . Jika harga sebuah HP merk A adalah Rp. 500.000,- lebih mahal dari harga HP merk B, maka harga sebuah HP merk A dan 3 HP merk B adalah ...​


Misalkan :

a = HP merk A

b = HP merk B

SPLDV

Persamaan 1 : 3a + 2b = 9.000.000

Persamaan 2 : a = b + 500.000

Substitusi persamaan 2 ke persamaan 1

3a + 2b = 9.000.000

3(b + 500.000) + 2b = 9.000.000

3b + 1.500.000 + 2b = 9.000.000

5b + 1.500.000 = 9.000.000

5b = 9.000.000 - 1.500.000

5b = 7.500.000

b = 7.500.000 ÷ 5

b = 1.500.000

Substitusikan b = 1.500.000 ke persamaan 2

a = b + 500.000

a = 1.500.000 + 500.000

a = 2.000.000

Jadi, harga sebuah HP merk A dan 3 HP merk B

= a + 3b

= 2.000.000 + 3(1.500.000)

= 2.000.000 + 4.500.000

= 6.500.000
